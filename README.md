### Some of my projects
Major
- [openascii](https://github.com/asciimoth/openascii) - collection of opensource ascii art (mostly mine) built around my own [animated ascii art](https://github.com/asciimoth/3a) format
  - [aaa](https://github.com/asciimoth/aaa) - cli tool to work with 3a format (renderer, editor, convertor, sys fetch, etc)
  - [rs3a](https://github.com/asciimoth/rs3a), [py3a](https://github.com/asciimoth/py3a), [go3a](https://github.com/asciimoth/go3a) libs
- [ygg](https://github.com/asciimoth/ygg) - alternative feature-rich [yggdrasil mesh network](https://github.com/yggdrasil-network/yggdrasil-go) implementation with [WASM builds supported](https://asciimoth.github.io/ygg/)
  - [yggpeers](https://github.com/asciimoth/yggpeers) - static yggdrasil public peers JSON generator for GitHub Pages
- [wgo](https://github.com/asciimoth/wgo) - [WireGuard](https://www.wireguard.com/) embeddable library with [amnesia obfuscation](https://github.com/amnezia-vpn/amneziawg-go) support and a bunch of QoL features
  - [wg-web-demo](https://github.com/asciimoth/wg-web-demo) - browser WASM demonstration of HTTP over WireGuard over socks-over-websocket using userspace TCP/IP stack
- [socksgo](https://github.com/asciimoth/socksgo) – the most complete, compatible, feature-rich, and extensible SOCKS library for Go

Minor
- [pmark](https://github.com/asciimoth/p-mark) - Go lib and tool for tagging processes for split routing and other purposes
- [pasta](https://github.com/asciimoth/pasta) - Headless Go framework for node-based graph editors and runtimes
- [mnlib](https://github.com/asciimoth/mnlib) - mesh-oriented DNS names lib
- [gonnect](https://github.com/asciimoth/gonnect) - my collection of network helper functions and common types that I use across my go projects
- [gonnect-netstack](https://github.com/asciimoth/gonnect-netstack) - gVisor's userspace TCP/IP stack integration for the gonnect-based projects
- [gonnect-vpn-example](https://github.com/asciimoth/gonnect-vpn-example) - an example of simple point-to-point VPN built on top of the gonnect "ecosystem"
- [tuntap](https://github.com/asciimoth/tuntap) - cross-platform TUN device library extracted from wireguard-go
- [batchudp](https://github.com/asciimoth/batchudp) - UDP transport package extracted from wireguard-go/conn
- [bufpool](https://github.com/asciimoth/bufpool) - byte buffer pool interface and testing helpers
- [putback](https://github.com/asciimoth/putback) - a minimal library that provides wrappers for common I/O interfaces, adding the ability to return read bytes back to the stream for subsequent reading
- [pineentry](https://github.com/asciimoth/pineentry) - GPG pinentry caching proxy
- [ident](https://github.com/asciimoth/ident) - RFC 1413 implementation
- [licensedb](https://github.com/asciimoth/licensedb) - database of SPDX licenses embedded to a go library
- [version](https://github.com/asciimoth/version) - multi-source semantic version management tool
- [inplace](https://github.com/asciimoth/inplace) - go library for non-destructive document's content modification
- [colorit](https://github.com/asciimoth/colorit) - a minimal go library for syntax highliting
- [wasm-examples](https://github.com/asciimoth/wasm-examples) - examples of written by hand wasm programs
- [configer](https://github.com/asciimoth/configer) - Config formats abstraction lib for go
